# Claude Code Plugin: Code

Specialized coding agents and workflow definitions for [Claude Code](https://code.claude.com/).

## Agents

This plugin provides several specialized agents designed for specific development tasks:

- **typescript-pro**: Expert TypeScript & Bun developer specializing in advanced type system usage and Bun-first development.
- **go-cli**: Expert Go developer focused on building modern CLI applications with Cobra and Viper.
- **spring-boot**: Expert Java/Kotlin developer specializing in Spring Boot microservices and modern backend patterns.
